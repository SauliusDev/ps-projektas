# Step 6: Generate — Write Artifacts

## Your Task

For each selected candidate: generate the full artifact content, ask where to save it (`l` local / `g` global), then write the file. Process one artifact at a time in the order the user selected them.

---

## Per-Artifact Loop

For each selected candidate, do the following in sequence:

### 1. Show full artifact preview

Generate the complete file content using the template for its type (see templates below). Display it in a code block.

### 2. Ask destination

```
Save "{artifact-name}" to:
  [l]  local   — ./.claude/{type}/{name}
  [g]  global  — ~/.claude/{type}/{name}
  [s]  skip

Enter l, g, or s:
```

**HALT — wait for input before writing.**

### 3. Write the file

Write to the resolved path. If the file already exists, show:
```
⚠️  {path} already exists.  [o] overwrite  [s] skip
```
**HALT — wait.**

Confirm each write with a checkmark line:
```
✓  Written: ./.claude/skills/framework-cookbook/SKILL.md
```
