## Coverage Gaps

The MagicDraw HTML 2.0 export does **not include** sequence diagrams or the package/architecture diagram. These diagram types are absent from the export's `diagrams.xml` registry entirely — they were either not modeled in this export snapshot, or MagicDraw HTML 2.0 does not export those types.

**Diagrams with NO pre-rendered image:**
1. Sequence diagrams (Sekų diagramos) — 0 of N exported
2. Package / component / architecture diagram — 0 of 1 exported

These must be generated from alternative sources (PlantUML `.puml` files in `_refs/export-og/` or `_refs/export-fixed/`).

## SVG / Vector Format Note

No SVG files are present anywhere in this export. All renders are raster JPG.

- **Cannot** be opened in Inkscape, Adobe Illustrator, or edited in-browser
- **Cannot** be zoomed infinitely without quality loss
- To get vector output: re-export from the original MagicDraw project with SVG format enabled
