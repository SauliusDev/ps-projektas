**3.3 Pasirinkta programavimo kalba, technologija**  

Kuriamai sistemai pasirinktos tokios programavimo kalbos ir technologijos, kurios yra patogios internetinės sistemos kūrimui bei leidžia atskirti naudotojo sąsają nuo serverio dalies.

**Programavimo kalbos:**  
Frontend: TypeScript / JavaScript  
Backend: Python  

**Technologijos:**  
Frontend: React  
Frontend kūrimo aplinka: Vite  
Backend: FastAPI  
Duomenų bazė: PostgreSQL  
API: REST  
Integracija: Google Maps API  

Tokie pasirinkimai buvo padaryti todėl, kad React leidžia patogiai kurti interaktyvią naudotojo sąsają, o FastAPI leidžia greitai ir aiškiai kurti serverio pusės API. PostgreSQL pasirinkta duomenų saugojimui, nes ši duomenų bazė yra patikima ir tinkama struktūrizuotiems duomenims. Google Maps API planuojama naudoti maršrutų sudarymui ir vietų atvaizdavimui žemėlapyje.
